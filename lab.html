Lab Experiment 1: Networking Basics and Its Hardware Components

1.1. Construct a table that consists of S.No, Networking/other commands, Functions (write it in your own words). Under the table, paste the screenshot for every command. Submit this assignment as a softcopy (Word format).

1.2. Write in your own words about the networking components and devices based on the discussion we had during class (theory and lab) or you can refer to the internet, books, or anywhere.

Note: Do not copy from anywhere. A photocopy of handwritten material should be submitted as a PDF.

2.2. Create a folder and make it accessible by all the users. Create a file and paste it in the folder (File named with your registration number).

2.3. The admin has to allocate some space (ex: 1 Gb) from his machine, after the network creation and allocate some space (ex: 10 Mb for each user) for each user. Now any user can have the access to allocated space anywhere from the network.

Lab Experiment 4: Error Detection and Correction Mechanisms

4.1 Let the user choose any one of the following mechanisms. Get the data bits and necessary inputs from the user, and generate the original bit to be sent using

  1. Longitudinal Redundancy Check (LRC)
  2. Vertical Redundancy Check (VRC)
  3. Cyclic Redundancy Check (CRC)
  4. Checksum

4.2. Get the data bits from the user and generate the original bit to be sent using Hamming code (Sender side). Similarly, get the original data from the user and check whether the data is error-free or not. Detect and correct the error if any occurs (Receiver side).

For guidelines to submit the assignment, refer to: https://sites.google.com/view/swe2002/lab

Lab Experiment 5: IP Addressing Classless Addressing

5.1 Get the IPV4 address (should accept any format such as Binary, dotted decimal, or hexadecimal) from the user and convert the IP address into any one of the formats (Binary, dotted decimal, hexadecimal) as the user described.

5.2 Prompt the user to enter the IP address, IP address type (IPV4 or IPV6), and number of subnetworks, identify the Class name (if it is classful addressing), Calculate the network ID, Host ID, subnet mask, total number of address spaces, first address, last address and total number of address for the same IP address using functions.

Note: For the above questions you have to Illustrate the process step by step with the help of photos that you have taken during the lab.
